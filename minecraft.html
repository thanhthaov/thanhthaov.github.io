MINECRAFT
The idea behind this Minecraft project was to explore a virtual, public, and online environment and learn what makes data into an object using a culturally relevant object.

This was my first time playing Minecraft but it was definitely a great learning experience. It took me many, many tries to survive the first night and also I was not very efficient about building this No Face structure. I chose No Face because I really enjoy Studio Ghibli films and animations because they were a large part of my childhood. If I had the time to go back and edit this, I would make it a lot bigger so that the mask features would be more refined and easily recognizable.
